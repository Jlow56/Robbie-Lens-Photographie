# J'apprends le développement Web avec HTML5 & CSS3

> « J'aimerais créer mon propre site web… mais je n’y connais rien… Par où commencer ? »

Vous êtes au bon endroit !  
Ce cours est **LE** cours à suivre si vous débutez complètement en programmation.  
Vous allez vous initier à deux langages accessibles, faciles à apprendre et incontournables du Web :

- **HTML5** : pour structurer le contenu de votre site (titres, paragraphes, liens, images, etc.)  
- **CSS3**  : pour mettre en forme et styliser ce contenu (couleurs, polices, mises en page, effets, etc.)

---

## Pourquoi HTML et CSS ?

**Séparation des responsabilités :**

1. **HTML**  
   Crée et structure le contenu.  
2. **CSS**  
   Gère la mise en forme visuelle.

Cette approche modulaire facilite la maintenance, l’accessibilité et l’évolution de votre site.

> *Sans CSS, votre site ressemblerait à un vestige des années 90 : sans styles, sans mise en page, sans dynamisme.*

---

## Organisation du cours

### Partie 1 : Bases du HTML

- Découverte des balises :
  - `<html>`
  - `<head>`
  - `<body>`
  - `<h1>` à `<h6>`
  - `<p>`
  - `<a>`
  - `<img>`
- Structuration du contenu et bonnes pratiques.

### Partie 2 : Bases du CSS

- Syntaxe des règles CSS (sélecteurs, propriétés, valeurs).  
- Liaison entre fichier HTML et fichier CSS.  
- Propriétés de style : couleurs, polices, bordures, arrière-plans.  
- Effets : ombres, transitions, interactions de base.

### Partie 3 : Mise en page avancée

- Structure d’une page : en-tête, contenu principal, sidebar, pied de page.  
- Techniques :
  - CSS Grid  
  - Flexbox  
- Utilisation d’une grille invisible pour disposer efficacement les blocs.

### Partie 4 : Responsive Design et éléments interactifs

- Media queries pour adapter le site à toutes les tailles d’écran.  
- Création de tableaux et de formulaires.  
- Validation de formulaires et interactions utilisateur.

---

## Objectifs pédagogiques

- **Maîtriser les bases de HTML5** : structuration sémantique et bonnes pratiques.  
- **Mettre en forme des pages web avec CSS3** : styles, positions, effets et animations.  
- **Agencer le contenu de pages web** : Grid, Flexbox, responsive design.  
- **Utiliser des fonctionnalités avancées** : formulaires, tableaux, transitions.
